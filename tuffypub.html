TuffyPub Github Project Link to TuffyPub Website

TuffyPub! TuffyPub is real-time web-based chat application developed in Angular 7 with Firebase as the database. Users can create an account, login and chat with other people in real-time. Users may chat in the global chat room or create their own chat rooms and also have the choice to make them private. User authentication provided through Firebase functions.
