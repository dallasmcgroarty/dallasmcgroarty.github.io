Boogle

Boogle is a Google search engine clone. Created using a LAMP stack along with Javascript/Jquery. Boogle allows the user to search for a term in the search bar. Boogle then retrieves the data of websites from the database that matches the term and displays it similar to how google does. Users may also search for images related to the search term. A PHP web crawler script is used to crawl the web for website information and stores it in the MySQL database.

Ranks

Ranks is a mobile first free score card web app. Users can add player cards which contain the player's name and score and dynamically update their score through some plus and minus buttons. No database used. Everything is strictly done using local storage. This way it is super light weight, fast, and non-invasive. Developed using HTML, CSS, Bootstrap, and JavaScript. If on iOS, disable private browsing in order to allow access to local storage.

Face Recognition

Web app created using React, Express, Node.js and PostgreSQL. The app uses the Clarifai API to detect faces in an image. The server and database are both deployed on Heroku, while the front-end is hosted on github pages. Once signed in or registered, enter the url of an image in the input space and click detect. The image will be displayed with a box around the face.

Members Only

A simple web application using PHP, Apache, MySQL, HTML, CSS, Bootstrap. Members Only allows users to securely register and sign in to a site and look at some data in a database. Once signed in and authenticated users have some access to the database and can view, add, or delete items in the database. SQL scripts are included to create the database, create a user to access the database, dump the database, and restore the database.

Tuffypub

TuffyPub is real-time web-based chat application developed in Angular 7 with Firebase as the database. Users can create an account, login and chat with other people in real-time. Users may chat in the global chat room or create their own chat rooms and also have the choice to make them private. User authentication provided through Firebase functions.

Robofriends

Web app created using React. Website displays randomly generated robots with their name and email and also includes a dynamic search function to filter the robots by name. Robot images, names, and emails are fetched from a website.

Open Inventory

Open Inventory is a web based inventory system designed to help people/businesses manage their product inventory. Go and MongoDB were used for the back-end, database and for user athentication.
