Projects

Boogle

Boogle is a Google search engine clone. Created using a LAMP stack along with Javascript/Jquery. Users may type a search term into the search bar to find relative websites, images, and news. Google Maps API allows users to search for locations as well. A PHP web crawler script is used to crawl the web for website information and stores it in the MySQL database.

Spodify

Spodify is a Spotify music player and storage clone. Created using a LAMP stack along with Javascript/Jquery. Users can create an account, browse available music, listen to music through the music player, and create their own playlists. The music player comes equipped with various funcionality like play next song, play previous song, shuffle songs, repeat songs, play and pause music, playback slider, and volume slider. Feel free to check it out now!

GetitDun

GetitDun is a simple and easy to use to-do list creator! Developed using Django, Python, HTML, and Bootstrap. This web app allows you to create new to-dos, view current to-dos, view completed to-dos, and update/delete current to-dos. Users must sign-up and login in order to start creating their to-dos.

DevConnector

DevConnector is a social platform for developers to create their own profile, share posts, comment, and connect. Created using MongoDB, Express, React and Node.js. Fully deployed on Heroku and available for use!

Ranks

Ranks is a mobile first free score card web app. Users can add player cards which contain the player's name and score and dynamically update their score through some plus and minus buttons. Developed using HTML, CSS, Bootstrap, and JavaScript. If on iOS, disable private browsing in order to allow access to local storage.

Face Recognition

Web app created using React, Express, Node.js and PostgreSQL. The app uses the Clarifai API to detect faces in an image. The server and database are both deployed on Heroku, while the front-end is hosted on github pages. Once signed in or registered, enter the url of an image in the input space and click detect. The image will be displayed with a box around the face.

Members Only

A simple web application using PHP, Apache, MySQL, HTML, CSS, and Bootstrap. Members Only allows users to securely register and sign in to a site and look at some data in a database. Once signed in and authenticated users have some access to the database and can view, add, or delete items in the database.
